Utopía

El proyecto utopía es un sistema integral de gestión educativo creado para cubrir las necesidades de aquellas entidades que manejan mucha carga académica, demanda estudiantil y se encuentran en sectores de bajos recursos y poca disponibilidad tecnológica.

Para el desarrollo de la parte visual, se usaron diferentes tipos de lenguajes, tales como JavaScript, HTML y CSS, creando así la aplicación bajo una tecnología llamada aplicaciones web progresivas (PWA).

Collegepw-Db

El proyecto (utopia-db) es un cliente de base de datos que busca comunicar bases de datos con un frontEnd (aplicación, programa), en pocas palabras un API.

Se creo este cliente para distribuir de forma optima el contenido de una manera eficaz, rápida y sencilla, así como también se pueden crear proyectos en base a él gracias a que esta echa bajo estándares de legibilidad y modularidad, teniendo de esta manera características de framework.
